# Rosebud ICE

2024-2026 US Government Actor Dataset — Position tracking, verification, and updates.

## Files

### `data/all_changes_2024_2026.csv`
92 officeholder transitions between 2024-2026. Each row tracks a former officeholder and their replacement.

### `data/new_politicians_2026.csv`
90 new politicians who entered Congress or Senate between 2024-2026.

### `data/updated_actors_2026.csv`
Full dataset (536 rows) with original columns preserved + 7 new verification columns.
